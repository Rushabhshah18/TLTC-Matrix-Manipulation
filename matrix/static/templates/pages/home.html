Example Section

Example 1 - Addition or Subtraction

eg. Matrix A:  |1  2|      Input: [1, 2, 4, 3]             Matrix B:   |2  4|          Input: [2, 4, 3, 1]
               |4  3|                                                  |3  1|

Matrix A details:                                            Matrix B details:
Row: 2     Columns: 2                                        Row: 2     Columns: 2

Results:
Addition:    |3  6|                                          Subtraction:    |-1  -2|
             |7  4|                                                          |1    2|

**Note: The rows and columns of Matrix A should be same for Matrix B.

Example 2 - Multiplication

eg. Matrix A:  |1  2|      Input: [1, 2, 4, 3]             Matrix B:   |2|          Input: [2, 3]
               |4  3|                                                  |3|

Matrix A details:                                            Matrix B details:
Row: 2     Columns: 2                                        Row: 2     Columns: 1

Results:
Multiplication:    |8 |
                   |17|

**Note: The rows and columns of Matrix A should be same for Matrix B.

Example 3 - Transpose

eg. Matrix A:  |1  2|      Input: [1, 2, 4, 3]
               |4  3|

Matrix A details:
Row: 2     Columns: 2

Results:
Transpose:    |1  4|
              |2  3|
