Problem Statement

Tutoring center is facing difficulty with helping students. There is one problem that decelerates their work. Matrix Operations are hard to solve and taking so much time, they need a program that addresses Matrix Operations quickly. This program will save time for tutors and students. There are four Matrix Operations need to be in this program. The first is Adding metrics; the second is Subtracting Matrix, the third is Multiplying by Another Metrics, the last is Transposing metrics. The program should have an option for the size of the Matrix, 2x2 and 3x3 and 4x4. The program provides four types of Matrix operations for the users. Then the users can determine the size of their Matrix and type the numbers in the Matrix format. The user only needs to click the enter button after they enter the numbers, then the calculation will be processed by the system. After that, the correct solution will be given on the user's screen.

Theory

Matrix addition and subtraction:

Addition of matrix (sum of a matrix) A + B is the operation of a finding of a matrix C, entry of which are equal to the sum of all corresponding entry of matrix A and B, can be defined as:

сij = aij + bij

Matrix multiplication:

The result of the multiplication of matrices Am×n and Bn×k the matrix Cm×k such that the element of the matrix C, standing in the i-th row and j-th column (cij), is equal to the sum of products of elements of the i-th row of the matrix A by the corresponding elements j-th column of matrix B:

cij = ai1 · b1j + ai2 · b2j + ... + ain · bnj

Transpose matrix:

Transposing the matrix is an operation on the matrix in which its rows and columns are swapped:

aT(ij) = a(ji)

Addition

The sum A+B of two m-by-n matrices A and B is calculated entrywise:
(A + B)i,j = Ai,j + Bi,j,
where 1 ≤ i ≤ m and 1 ≤ j ≤ n.

 
|3   8|  +   |4    0|  =  |7    8|
|4   6|      |1   -9|     |5   -3|
Multiplication

The product cA of a number c (also called a scalar in the parlance of abstract algebra) and a matrix A is computed by multiplying every entry of A by c:
(cA)i,j = c · Ai,j.

 
|1   2   3|  x  |7     8|  =  |58   64|

Transposition

The transpose of an m-by-n matrix A is n-by-n matrix AT (also denoted Atr or tA) formed by turning rows into columns and vice versa:
(AT)i,j = Aj,i.

                  |24    9|
Subtraction

A matrix can only be added to (or subtracted from) another matrix if the two matrices have the same dimensions.
